# Java Tic Tac Toe (GUI Version)

A classic Tic Tac Toe game built with **Java** and **Swing**. This project was developed as part of a multi-day coding challenge to transition from a console-based application to a fully functional Graphical User Interface (GUI).

## 🚀 Features
- **Dynamic Player Setup:** Custom names and symbol selection (X or O) for Player 1.
- **Persistent Score Tracking:** Keeps track of wins for both players across multiple rounds.
- **Smart Turn Logic:** The loser of the previous round (or the other player in case of a draw) starts the next game.
- **Winning Animation:** Highlights the winning row with a blinking effect.
- **User-Friendly GUI:** Clean layout with a status bar showing the current player and score.
- **Game Over Summary:** Displays the final results and a total winner announcement when exiting.

## 🛠️ Technologies
- **Language:** Java
- **Framework:** Swing (GUI)
- **Logic:** Custom 2D-Array based game logic with separate Backend/Frontend structure.

## 🕹️ How to Run
1. Ensure you have **Java (JDK)** installed on your machine.
2. Clone the repository:
   https://github.com/AndreasB1981
3. Compile and run the `TicTacToeGUI.java` file.

## 📝 Learning Journey
This project demonstrates the transition from procedural console logic to **event-driven programming**. Key concepts covered include:
- Object-Oriented Programming (Separation of Logic and UI)
- ActionListeners and Event Handling
- Multi-threading with `javax.swing.Timer` for animations
- Layout Management (BorderLayout, GridLayout)
